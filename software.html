Software Projects

Ranging from school assignments to software developed for a company.

Automated English to ASL

For my senior project I created a Python script that takes in (limited) english words and concatenates American Sign Language signs for the 3d interpreter to showcase. Although this is a proof of concept, if implemented outside of the Maya 3D program, I think this can be a tool for people to learn more about ASL and providing greater accessibility for the Deaf community.

Administrative Web App

Myself and 2 other developers created this MEAN (Mongo, Express, AngularJs, NodeJs) stack application to help Motorola Solutions organize their summer mentorship program.

FOMO App

FOMO was an app created in school to help students find new clubs and events going around campus. It was crated with Javascript and PHP

OpenGL Rendering

In order to learn more about the core structure of light and scene rendering in computer graphics, I created multiple surfaces through OpenGL (using C) that could be animated based off user input (moving the bus). I also created lamps with local lights by implementing fast decay rates to appear similar to a real street lamp. Most of this is done automatically in higher level programming languages.

IoT Project

Internet of things (IoT) is a new field in which creating smart devices connected (mainly wirlessly) to the internet can now provide better support to previous applications that were used without any internet connection. Myself and two electrical engineers created a prototype "Smart Stick" device that uses Google Directions API (Python script) within a Rasberry Pi and smartphone to trigger vibrations in a blind user's stick/pole. I was responsible for the software portion of the project - communication between devices and implementing the direction algorithm and API.

Java Swing Game

Feline Defense is a purely educational game with the purpose of improving my object oriented programing and understanindg bare bones game mechanics. This built my foundation of how games can be developed and expanded my knowledge of GUI design in Java. Even though Java Swing is somewhat outdated, the educational components I enlisted when creating this app has been transferable in other object oriented projects.
